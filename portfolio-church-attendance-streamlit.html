Project Overview

The Church Attendance System was built to simplify how members mark attendance during Sunday services. The goal was to reduce the workload on ushers and enable seamless record-keeping for the Teenage Church.

The system ensures fairness and accountability by preventing duplicate submissions, separating records for First and Second Service, and storing data securely in JSON format.

It was my first time working with streamlit too, so I did project-based learning to get the grasp of how streamlit functions, it was really really fun, I had fun doing it.
